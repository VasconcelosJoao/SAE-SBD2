pergunta:"Indique qual a expressão mais adequada para identificar o atributo (ou atributos) que é  usado por um Índice para tornar mais eficiente uma consulta no Sistema Gerenciador de Banco de Dados."
resposta:"Chave de Procura"
pergunta:"Quantos tipos de Índices Ordenados foram estudados para compreensão do funcinamento mais eficiente desse recurso na elaboração de um Projeto Físico de Banco de Dados?"
resposta:"Existem dois tipos para os índices ordenados (densos e esparsos)."
pergunta:"Com base nas definições iniciais de Índice para Banco de Dados podemos afirmar que:"
resposta:"Um índice irá consistir de uma entrada para cada registro do arquivo, sendo que as entradas encontram-se ordenadas pelo valor da chave de acesso."
pergunta:"Qual das instruções SQL estaria criando um índice para o atributo idade (inteiro) na tabela CLIENTE?   Selecione a(s) opção(ões) que estiver(em) correta(s)."
resposta:"CREATE INDEX Cliente_Idade_IDX ON CLIENTE(idade); CREATE UNIQUE INDEX Cliente_Idade_IDX ON CLIENTE(idade);"
